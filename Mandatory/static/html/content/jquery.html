JQuery

Short introduction to Jquery.

The purpose of JQuery is to simplify coding with Javascript. It is basically a an enormous Javascript library with code for HTML, CSS, AJAX etc.
There are of course other Javascript libraries, but JQuery is by far one of the popular ones. JQuery is also cross-platform compatible and supports various browsers.

Implementation

In order to use JQuery in our code we must first implement it. JQuery is applied by implementing the JQuery script, which has multiple providers. One of these providers are Google.
Once the script is acquired, it is placed in the bottom of the body tag and then JQuery should work.
To test this, you can write a JQuery script and see if it appears as intended: 

    windows.onload = function() = {
        if(window.jQuery) {
            alert('Works')
        } else {
            alert('Doesn't work')
        }
    }

Another way of checking if JQuery is applied is to write the following inside the Javascript file:

    console.log($)
