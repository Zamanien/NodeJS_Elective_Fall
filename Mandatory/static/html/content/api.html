REST API

Short introduction to general API & REST-API.

API = Application Programming Interface
Allows one piece of software to communicate with another piece of software.
or
A type of contract provided by one piece of software to another piece of software.
Acts as the messenger between two entities.
F.x. Guest —> Waiter (API) —> Kitchen

"API is a UI with different users (Systems & Computers) in mind."

What is REST?
REpresentational State Transfer
REST is a set of rules/standards/guidelines (REST architecture) for building a Web API. On top of this, *REST API refers to tools, services or software that is based on the REST architectural principle.

What is RESTful Web Service?
RESTful Web Services are web services that obey the REST architecture style / Infrastructure. The function of the RESTful Web Service is to share data through an API. With RESTful Web Services you can either make your program into an API - Accessible for others or you can access third party API's & and get required Data.
