NodeJS Intro

It is always good to explain what exactly we are dealing with. So here is a short introduction:

NodeJS

NodeJS is basically a way of running JavaScript code. Before Node, JavaScript could only be applied to web browsers. By integrating a JavaScript engine (Chrome V8) with C++, the JavaScript code can now be applied outside browsers. Node is ideal for building highly scalable, data-intensive backend & real-time apps. Being great for prototyping & agile gives Node an advantage over the alternatives, such as Django. Node is fast & used world-wide. Source code will also be more clean & consistent, as there is JavaScript in both on the frontend & backend.


Node is a runtime environment of executing JavaScript code

Not a programming language. Neither a Framework.


What can you do with NodeJS?
  • API (RestAPI)
  • Websites
  • Standalone software
  • Servers
  • Comandlines
  • Scripts (timed scripts, etc)
  • Interfaces
